# About
<a href="https://metanote.org/" target="_blank">MetaNote.org</a>は、数少ない<b>日本語の論文投稿サイト</b>です。

「査読不要・審査なし」を原則とし、誰でも簡単にアイディアを投稿できます。

また、論文の共同編集や寄付機能も実装しています。
